Objects and Classes

Objects are collections of key-value pairs that store data and behavior (methods). They can be created using object literals ({}) or constructors. 

const guitar = {
    make: "Gibson", // Brand of the guitar
    model: "Les Paul", // Model name
    drive: function() { // Method that simulates playing the guitar
        console.log("Shredding...");
    }
};

Classes are templates for creating multiple objects with shared properties and methods. They provide a structured way to define object blueprints. 

class Guitar {
    /**
     * Creates a new Guitar instance.
     * @param {string} make - The brand of the Guitar (e.g., "Gibson").
     * @param {string} model - The model of the Guitar (e.g., "SG").
     */
    constructor(make, model) {
        this.make = make; // Assigns the brand to the object
        this.model = model; // Assigns the model to the object
    }

    // Method to simulate playing the guitar
    shred() {
        console.log("Shredding...");
    }
}

// Creating instances of the Guitar class
const myGuitar1 = new Guitar("Gibson", "Les Paul"); // Gibson guitar instance
const myGuitar2 = new Guitar("Fender", "Stratocaster"); // Fender guitar instance

Key Differences:

  • Objects are instances with their own properties and methods.
  • Classes define the blueprint for creating multiple objects with a consistent structure.
  • Objects can be created ad hoc, while classes allow reusable and scalable object creation.
